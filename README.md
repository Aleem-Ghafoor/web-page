# Web Page

A simple responsive personal portfolio practice website built with HTML and CSS.

## Pages

- `index.html` - Home page
- `about.html` - About page
- `projects.html` - Projects page

## Features

- Responsive header and footer
- Personal introduction section
- About and projects pages
- Image assets and custom CSS styling

## How to Open

Open `index.html` in a browser, or run the folder with a local live server.
